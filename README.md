# 🚀 Multi-Agent Agentic AI System

### Autonomous Data Analysis Pipeline

---

## 📌 Overview

This project is an **end-to-end autonomous Agentic AI system** where multiple intelligent agents collaborate under a central orchestrator to perform data analysis tasks — from data acquisition to insight generation and reporting — with minimal human intervention.

---

## 🎯 Features

* 🤖 Multi-agent architecture with intelligent coordination
* 📥 Automatic data acquisition (if dataset not provided)
* 🧹 Data cleaning and preprocessing
* 📊 Exploratory Data Analysis (EDA)
* ⚙️ Feature engineering and transformation
* 📈 Statistical analysis and insights
* 💡 Human-readable insight generation
* 📑 Automated reporting and visualization
* the frontend is in the process of enhancement

---

## 🧠 Architecture Overview

The system consists of multiple specialized agents:

* **Orchestrator Agent** – Manages workflow and agent coordination
* **User Interaction Agent** – Understands user requirements
* **Data Acquisition Agent** – Fetches datasets automatically
* **Data Understanding Agent** – Profiles and inspects data
* **Data Cleaning Agent** – Handles missing values and inconsistencies
* **EDA Agent** – Performs analysis and visualization
* **Feature Engineering Agent** – Prepares data for modeling
* **Statistical Analysis Agent** – Performs hypothesis testing
* **Insight Generation Agent** – Converts results into insights
* **Reporting Agent** – Generates dashboards and reports

---

## ⚙️ Tech Stack

This project is built using a modern full-stack and agentic AI architecture:

* 🧠 **Google ADK** – for building and managing the multi-agent system
* ⚡ **FastAPI** – for high-performance backend APIs
* 🎨 **React** – for interactive frontend interface
* 🐍 **Python** – core language for agent logic and data processing
* 📊 **Pandas, NumPy, Scikit-learn** – data analysis and modeling
* 📈 **Matplotlib / Plotly** – data visualization


## ▶️ How to Run

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/agentic-ai-system.git
cd agentic-ai-system
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run Backend (FastAPI)

```
uvicorn backend.app:app --reload
```

### 4️⃣ Run Frontend (React)

```
cd frontend
npm install
npm start
```

## 📊 Example Workflow

1. User provides problem statement

4. Cleans and analyzes data
5. Generates insights
6. Outputs visual report/dashboard
